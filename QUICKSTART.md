## 📝 Быстрый старт

### 1. Настройте дату свадьбы

Откройте следующие файлы и измените дату (в начале каждого компонента):

**Мобильная версия:** `src/imports/СайтПриглашениеНаСвадьбуМобилка/СайтПриглашениеНаСвадьбуМобилка.tsx` (строка ~7)
```typescript
const weddingDate = new Date('2026-08-30T16:00:00'); // Измените на вашу дату
```

**Десктопная версия:** `src/imports/СайтПриглашениеНаСвадьбуДесктоп/СайтПриглашениеНаСвадьбуДесктоп.tsx` (строка ~8)
```typescript
const weddingDate = new Date('2026-08-30T16:00:00'); // Измените на вашу дату
```

### 2. Опубликуйте на GitHub Pages

**Самый простой способ:**

1. Создайте репозиторий на GitHub
2. Перейдите в Settings → Pages
3. В "Build and deployment" выберите "GitHub Actions"
4. Загрузите проект в репозиторий:
   ```bash
   git init
   git add .
   git commit -m "Wedding invitation website"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO_NAME.git
   git push -u origin main
   ```
5. Подождите 1-2 минуты — ваш сайт готов!

**Адрес сайта:** `https://USERNAME.github.io/REPO_NAME/`

### 3. Настройте базовый путь (важно!)

Если ваш репозиторий называется не `username.github.io`, обновите `vite.config.ts`:

```typescript
export default defineConfig({
  base: '/REPO_NAME/',  // Добавьте имя вашего репозитория
  plugins: [react(), tailwindcss()],
  // ...
})
```

Затем:
```bash
git add vite.config.ts
git commit -m "Configure base path"
git push
```

## 🔧 Дополнительные настройки

### Изменить текст или дизайн

Текст и дизайн находятся в файлах:
- `src/imports/СайтПриглашениеНаСвадьбуМобилка/СайтПриглашениеНаСвадьбуМобилка.tsx`
- `src/imports/СайтПриглашениеНаСвадьбуДесктоп/СайтПриглашениеНаСвадьбуДесктоп.tsx`

### Использовать собственный домен

1. Создайте файл `public/CNAME` с вашим доменом:
   ```
   wedding.example.com
   ```
2. В DNS добавьте CNAME запись на `username.github.io`
3. В GitHub Settings → Pages укажите ваш домен
