Hero блок (первый блок с билетом-приглашением) теперь полностью соответствует макетам из Figma в обеих версиях.

## 🎨 Десктопная версия (≥ 800px)

**Файл:** `src/imports/СайтПриглашениеНаСвадьбуДесктоп/Section.tsx`

### Особенности:
- ✅ Билет повернут на **-2.19 градусов** (небольшой наклон)
- ✅ Горизонтальное расположение контента
- ✅ Пунктирная вертикальная линия разделяет информацию
- ✅ SVG фон с тенью (shadow)
- ✅ Левая часть: точки, текст "Приглашение на свадебное торжество", разделитель, имена, декоративный элемент
- ✅ Правая часть: дата и время

### Структура:
```
Section (контейнер)
└── Ticket (билет с поворотом -2.19°)
    ├── Bg (SVG фон с тенью)
    └── Content
        ├── Container (левая часть - имена)
        └── Container10 (правая часть - дата/время)
```

## 🔧 Как использовать

Hero блок автоматически импортируется в основных компонентах:

**Десктоп:**
```typescript
import HeroSection from "./Section";

<section className="h-[400px] relative shrink-0 w-full">
  <HeroSection />
</section>
```

**Мобилка:**
```typescript
import HeroSection from "./Section-2-129";

<section className="h-[816px] relative shrink-0 w-full">
  <HeroSection />
</section>
```
