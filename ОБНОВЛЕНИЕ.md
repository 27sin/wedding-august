## Изменение даты свадьбы

Дата свадьбы указана в начале каждого файла компонента:

1. Откройте файл мобильной версии:
   ```
   src/imports/СайтПриглашениеНаСвадьбуМобилка/СайтПриглашениеНаСвадьбуМобилка.tsx
   ```
   Найдите строку ~7:
   ```typescript
   const weddingDate = new Date('2026-08-30T16:00:00');
   ```

2. Откройте файл десктопной версии:
   ```
   src/imports/СайтПриглашениеНаСвадьбуДесктоп/СайтПриглашениеНаСвадьбуДесктоп.tsx
   ```
   Найдите строку ~8:
   ```typescript
   const weddingDate = new Date('2026-08-30T16:00:00');
   ```

3. Измените дату в формате `YYYY-MM-DDTHH:MM:SS`
   - `YYYY` — год (например, 2026)
   - `MM` — месяц (01-12)
   - `DD` — день (01-31)
   - `HH` — часы (00-23)
   - `MM` — минуты (00-59)
   - `SS` — секунды (обычно 00)

**Примеры:**
- 15 июня 2026, 18:00 → `2026-06-15T18:00:00`
- 1 сентября 2027, 15:30 → `2027-09-01T15:30:00`

## Изменение текстов

Все тексты находятся в тех же файлах. Просто найдите нужный текст и измените его:

### Имена молодоженов
Найдите строки с `Максим` и `Оксана` — замените на свои имена.

### Место проведения
Найдите `Ресторан "Шале"` и адрес — замените на своё место.

### Программа дня
Найдите массив с временами и описаниями:
```typescript
{[
  { time: "16:00", title: "Сбор гостей", desc: "Приветственные напитки и фотосессия" },
  { time: "16:30", title: "Церемония", desc: "Самый трогательный момент" },
  // ...
]}
```

### Цветовая палитра
Найдите массив с цветами:
```typescript
{[
  { color: "#d9d9d9", name: "Светло-серый" },
  { color: "#dbcab6", name: "Бежевый" },
  // ...
]}
```

## Замена изображений

Изображения находятся в папках:
- Мобильная: `src/imports/СайтПриглашениеНаСвадьбуМобилка/`
- Десктопная: `src/imports/СайтПриглашениеНаСвадьбуДесктоп/`

Чтобы заменить изображение:
1. Сохраните новое изображение с **тем же именем** (например, `9450b4cf47f57ad9e8dcb2c3b4733bf49d484dd0.png`)
2. Или измените импорт в начале файла:
   ```typescript
   import imgImage14 from "./ваше-изображение.png";
   ```

## Добавление ссылок на кнопки

### Кнопка "Заполнить анкету"
Найдите:
```typescript
<button className="bg-[#525834] text-white ...">
  Заполнить анкету
</button>
```

Замените на:
```typescript
<a href="https://ваша-ссылка-на-форму.com" target="_blank" rel="noopener noreferrer"
   className="inline-block bg-[#525834] text-white ...">
  Заполнить анкету
</a>
```

### Кнопка "Вступить в чат"
Аналогично замените:
```typescript
<a href="https://t.me/ваш_чат" target="_blank" rel="noopener noreferrer"
   className="border border-[#525834] ...">
  ...
  Вступить в чат
</a>
```

### Кнопка "Открыть в Яндекс.Картах"
Замените:
```typescript
<a href="https://yandex.ru/maps/..." target="_blank" rel="noopener noreferrer"
   className="w-full bg-[#525834] ...">
  Открыть в Яндекс.Картах
</a>
```

## Интеграция карты Яндекс

Найдите:
```typescript
<div className="bg-gray-200 rounded-lg h-80 ...">
  <p className="text-gray-500">Карта Яндекс.Карты</p>
</div>
```

Замените на реальную карту Яндекс:
```typescript
<iframe
  src="https://yandex.ru/map-widget/v1/?..."
  className="w-full h-80 rounded-lg"
  frameBorder="0"
  allowFullScreen
/>
```

Получить код карты можно на [Яндекс.Картах](https://yandex.ru/map-constructor/).

## Проверка изменений

После внесения изменений:
1. Сохраните файлы
2. Откройте превью в браузере
3. Проверьте мобильную версию (уменьшите окно < 800px)
4. Проверьте десктопную версию (увеличьте окно ≥ 800px)

## Публикация изменений

После внесения всех изменений:
```bash
git add .
git commit -m "Обновлен контент сайта-приглашения"
git push
```

GitHub Actions автоматически опубликует обновленный сайт!
