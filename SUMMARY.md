## 🚀 Быстрый старт

### 1. Настройте дату свадьбы
Измените в обоих файлах (строка ~7-8):
- `src/imports/СайтПриглашениеНаСвадьбуМобилка/СайтПриглашениеНаСвадьбуМобилка.tsx`
- `src/imports/СайтПриглашениеНаСвадьбуДесктоп/СайтПриглашениеНаСвадьбуДесктоп.tsx`

```typescript
const weddingDate = new Date('2026-08-30T16:00:00');
```

### 2. Опубликуйте на GitHub
```bash
git init
git add .
git commit -m "Wedding invitation website"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

### 3. Включите GitHub Pages
Settings → Pages → Source: "GitHub Actions"

### 4. Настройте базовый путь (если нужно)
В `vite.config.ts`:
```typescript
base: '/REPO_NAME/',
```
