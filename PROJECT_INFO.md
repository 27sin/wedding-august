# 💍 Информация о проекте

## 🎯 Что это?

Полностью готовый сайт-приглашение на свадьбу, созданный на основе дизайна из Figma.

## ✨ Особенности

### Адаптивный дизайн
- **Мобильная версия:** для экранов < 800px
- **Десктопная версия:** для экранов ≥ 800px
- Автоматическое переключение между версиями

### Функциональность
- ✅ Живой счетчик обратного отсчета до свадьбы (обновляется каждую минуту)
- ✅ Элегантный дизайн с использованием шрифтов Playfair Display и Open Sans
- ✅ Все секции: дата, место, программа, дресс-код, важные детали
- ✅ Карта Яндекс (placeholder - можно заменить на реальную)
- ✅ Кнопки для заполнения анкеты и вступления в чат Telegram

### Готовность к публикации
- ✅ Настроен для GitHub Pages
- ✅ Автоматическое развертывание через GitHub Actions
- ✅ .nojekyll для корректной работы
- ✅ Оптимизированная сборка

## 🏗️ Технический стек

- **React 18** — библиотека для построения UI
- **TypeScript** — типизация
- **Vite** — сборщик
- **Tailwind CSS v4** — стилизация
- **Google Fonts** — шрифты

## 🔄 Как обновить контент

### Дата свадьбы
Файлы:
- `src/imports/СайтПриглашениеНаСвадьбуМобилка/СайтПриглашениеНаСвадьбуМобилка.tsx` (строка ~1547)
- `src/imports/СайтПриглашениеНаСвадьбуДесктоп/СайтПриглашениеНаСвадьбуДесктоп.tsx` (строка ~1567)

Найдите и измените:
```typescript
const weddingDate = new Date('2026-08-30T16:00:00');
```

### Текст и контент
Все тексты находятся в соответствующих tsx-файлах в папке `src/imports/`.
Просто найдите нужный текст и измените его.

### Изображения
Изображения из Figma уже импортированы и находятся в папках:
- `src/imports/СайтПриглашениеНаСвадьбуМобилка/*.png`
- `src/imports/СайтПриглашениеНаСвадьбуДесктоп/*.png`

Чтобы заменить изображение, просто замените файл с тем же именем.

## 🌐 Публикация

### Минимальная настройка (5 минут)
1. Создайте репозиторий на GitHub
2. Загрузите проект
3. Включите GitHub Pages (Settings → Pages → GitHub Actions)
4. Готово!

### С базовым путем
Если репозиторий не `username.github.io`, обновите `vite.config.ts`:
```typescript
base: '/repo-name/'
```

### С собственным доменом
Создайте `public/CNAME` с вашим доменом и настройте DNS.
